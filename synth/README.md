# Synthetic Materials

Practice files for WiFi penetration testing without live targets.

## Directory Structure

```
synth/
├── beginner/
│   ├── captures/      # Sample pcap files
│   ├── hashes/        # Hashcat practice files
│   └── wordlists/     # Curated wordlists
├── intermediate/
│   ├── captures/      # EAP/PMKID captures
│   ├── hashes/        # Enterprise hashes
│   └── configs/       # hostapd-mana configs
└── advanced/
    ├── captures/      # WPA3/DragonBlood samples
    └── scripts/       # SDR fuzzing templates
```

## Usage

1. Use capture files with Wireshark/tshark for analysis practice
2. Use hash files with hashcat for cracking practice
3. Use wordlists for password recovery exercises

## Safety

All files are synthetic - no real network data included.
